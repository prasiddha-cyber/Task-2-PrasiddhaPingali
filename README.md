# ⚗️ Cipher Lab – Encryption & Decryption Workbench

A modern desktop application developed as **Task 2** for the **DecodeLabs Cyber Security Internship 2026**. The application demonstrates classical cryptographic techniques through an interactive workbench that allows users to encrypt, decrypt, and analyze text using multiple cipher algorithms.

---

## 🚀 Features

- 🔐 Encrypt plain text using classical ciphers
- 🔓 Decrypt encrypted messages
- 🔤 Caesar Cipher
- 🔑 Vigenère Cipher
- 🔄 Atbash Cipher
- 🔢 ROT13 Cipher
- 🔡 Character transformation analysis
- 📊 Encryption metrics dashboard
- 📋 Security information panel
- 📄 Export results and reports
- 🖥️ Modern desktop GUI with intuitive navigation

---

## 🛠️ Technologies Used

- Python 3
- CustomTkinter
- Tkinter
- String Processing
- Classical Cryptography Algorithms
